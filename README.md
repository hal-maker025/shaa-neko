# シャーシャー猫 🐈

通話相手の怒鳴り声を、猫の威嚇（シャー！）に変換するアプリです。

大きな声・キツい口調の通話が続くと地味に消耗しますが、
音量が上がった瞬間に猫が「シャー！」と身構えてくれるので、
場の空気を少しやわらげたり、自分の気持ちを切り替えるきっかけになります。

音声は録音・送信されず、**ブラウザ内で音量レベルの解析のみ**を行います。
（内容は一切見ていません／どこにも送信していません）

## 使い方

### Web版（インストール不要・ブラウザだけで使える）

1. 公開ページ（`https://あなたのユーザー名.github.io/shaa-neko/`）を開く
2. 「🎙 マイクをオンにする」を押す → ブラウザがマイクの使用許可を求めるので「許可」
3. 通話（Zoomなど）をスピーカーで再生しながら使うと、相手の声にも反応します
4. 感度スライダーで、どれくらいの声量で「シャー」と反応するか調整できます
5. 「👆 タップでシャー」で動作確認、右上のトグルで効果音のON/OFFも可能
6. 「本日の威嚇回数」で、その日どれくらい猫が身構えたかがわかります

### Chrome拡張版（通話タブの音声を直接ひろえる）

マイク経由よりも正確に、通話ソフトのタブ音声を直接解析できます。

1. Chromeで `chrome://extensions` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」→ `extension` フォルダを選択
4. ツールバーの猫アイコンをクリック → サイドパネルが開く
5. 通話ソフト（ブラウザ電話・Zoom Web版など）のタブを開いた状態で
   「🎧 このタブの音声をひろう」を押すと、そのタブの音声で猫が反応します
   （取得中も相手の声はそのまま聞こえます）

## フォルダ構成

```
shaa-neko/
├── docs/          … Web版（GitHub Pagesでそのまま公開できます）
│   └── index.html
└── extension/     … Chrome拡張版（サイドパネル + タブ音声キャプチャ）
    ├── manifest.json
    ├── background.js
    ├── sidepanel.html
    ├── sidepanel.js
    └── icons/
```

## 注意事項

- タブ音声キャプチャはPCのChrome / Edge向けの機能です
- `chrome://` ページやウェブストアのページの音声は取得できません
- 職場で使う場合は、画面上のツール利用ルールを一応ご確認ください
